# Cerevolt – Revolution for the Mind
AI-Powered Cognitive & Emotional Support Platform for Children (5–12 Years)

---

##  Project Overview

Cerevolt is an AI-driven emotional and cognitive support platform designed to assist children aged 5–12 facing attention deficits, overstimulation, emotional dysregulation, and autism spectrum challenges.

The system integrates therapeutic media, gamified neurocognitive training, adaptive AI personalization, and optional neurofeedback support across Home, School, and Clinical environments.

---

##  Functional Requirements

### 1. User Management
- Role-based login (Admin, Parent, Teacher, Clinician, Student)
- Secure authentication & authorization
- Multi-user dashboard support
- Parental consent verification

---

### 2. Child Interface
- Screen-time controlled sessions
- Gamified cognitive exercises
- Slow-media therapeutic videos
- Audio therapy (binaural beats, calming stories)
- Learning modules
- Activity tracking
- Attention and cognitive scoring
- Area of concern detection
- Personalized recommendations

### 3. Parent Interface

- Emotional trend tracking
- Risk diagnosis insights

- Book consultants
- Offline practice kit access
- Medical profile matching

### 4. School Interface

- Student performance reports

### 5. Clinical Interface

- Neurofeedback integration (EEG/HRV optional)

### 6. Admin Panel
